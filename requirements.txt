# ============================================
# 安装说明
# ============================================
# 1. 创建虚拟环境: python -m venv synenv
# 2. 激活虚拟环境:
#    - Windows: synenv\Scripts\activate
#    - Linux/Mac: source synenv/bin/activate
# 3. 安装依赖: pip install -r requirements.txt
#
# 注意事项:
# - 无版本号，保持适应不同 Python 版本
# - 已移除不需要的库（torch, transformers, datasets, huggingface-hub）
# - 建议使用 Python 3.11+
# - 浏览器已存在于 .playwright-browsers 和 .chrome-for-testing，无需重新安装
#
# 重要提示:
# - crawl4ai 可能与 biliup 版本冲突（aiohttp 版本要求不同）
# - 如遇冲突，建议在独立环境安装 crawl4ai
